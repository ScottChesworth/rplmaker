rplMaker

Convert plugin preset files into a REAPER preset library (RPL) that loads through REAPER's accessible preset combobox.

Step 1: Template

In REAPER, save any one preset for the plugin from its FX preset combobox, then choose "Export preset library" from the preset menu. That exported RPL is the template: it teaches rplMaker the plugin-specific data it can't learn from the preset files alone.

Step 2: Presets to convert

Scanning digs factory presets out of the plug-in file itself, for plug-ins that ship no preset files. It needs the template from step 1 first, and found presets get numbered names since real names are not stored in a readable form.

      Keyboard help
      • Shift or Control with arrows: select more than one preset
      • Control+Up or Control+Down: move the selected presets
      • Control+Home or Control+End: move the selection to the top or bottom
      • Control+X then Control+V: cut the selected presets, then drop them below the focused preset
      • Alt+Up or Alt+Down: seek to the start of the previous or next folder
      • Shift+F: select the whole folder around the current preset
      • Control+F: find a preset by name; Enter searches, Shift+Enter searches backwards
      • F3 or Shift+F3: repeat the search forwards or backwards
      • F2: rename the selected preset; Enter applies, Escape cancels
      • Delete: remove the selected presets
      • Control+S: save changes
      • F1: open this help; Escape closes it and returns to the list
